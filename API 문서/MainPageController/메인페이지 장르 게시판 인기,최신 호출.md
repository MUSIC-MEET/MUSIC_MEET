# 장르 게시판 인기,최신.md
### API명 : `/boards/{genre}/{type}/{page}`

### method : GET

### comment : 메인 페이지에 장르게시판_약식을 호출하는 API genre에는 장르, type에는 popular(좋아요), latest(최신) 등의 분류, page는 가져올 페이지숫자를 넣어주면 된다.

### request :
    NULL

### response :
~~~json
{
    "id" : "1",
    "user" : "전국노예자랑",
    "title" : "이번 노래 좋네요"
}
~~~
### status code
가능 : 200 (HttpStatus.OK)

불가능 : 400 (HttpStatus.BAD_REQUEST)
