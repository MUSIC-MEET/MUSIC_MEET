# 메인페이지 유저 업로드 최신.md
### API명 : `/cover/latest/{num}`

### method : GET

### comment : 메인 페이지에 표시할 유저 업로드를 최신 순으로 호출 num 값으로 가져올 갯수 입력

### request :
    Null

### response :
~~~json
{
    "id" : "", // musicNum
    "imgSrc" : "", // 경로
    "title" : "", // origin_title
    "user" : "" // 닉네임
}
~~~
### status code
가능 : 200 (HttpStatus.OK)
