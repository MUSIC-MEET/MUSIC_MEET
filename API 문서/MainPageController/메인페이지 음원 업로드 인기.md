# 메인페이지 음원 업로드 인기.md
### API명 : `/music/popular/{num}`

### method : GET
 
### comment : 메인 페이지에 표시할 음원을 좋아요 순으로 호출 num 값으로 가져올 갯수 입력

### request :
    Null

### response :
~~~json
{
    "id" : "", // musicNum
    "imgSrc" : "", // 경로
    "title" : "", // origin_title
    "artist" : "" // origin_singer
}
~~~
### status code
가능 : 200 (HttpStatus.OK)
