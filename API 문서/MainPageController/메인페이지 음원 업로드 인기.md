# 메인페이지 음원 업로드 인기.md
### API명 : `/music/popular/{num}`

### method : GET
 
### comment : 메인 페이지에 표시할 음원을 좋아요 순으로 호출 num 값으로 가져올 갯수 입력

### request :
    Null

### response :
~~~json
{
    "id" : "1",
    "imgSrc" : "localhost:8080/music/image/좋은날.png",
    "title" : "좋은날",
    "artist" : "아이유(IU)"
}
~~~
### status code
가능 : 200 (HttpStatus.OK)
