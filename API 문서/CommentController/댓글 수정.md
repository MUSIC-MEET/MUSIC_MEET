# API 제목
### API명 : `/board/comment`

### method : PUT

### comment : 게시판에 달린 댓글을 수정하는 API

### request :
~~~json
{
    "genre" : "kpop",
    "commentNum" : "2",
    "content" : "ㅇㅇㅇㄴㅁㅇㅁㅇㅈㅇㄴㅁㅇㅈㅁ"
}
~~~

### response :
    NULL


### status code
인증 실패 : 401 (HttpStatus.UNAUTHORIZED)

수정 성공 : 204 (HttpStatus.NO_CONTENT)

// 성공을 제외한 예외처리
수정 실패 : 400 (HttpStatus.BAD_REQUEST)
