# 댓글 삭제
### API명 : `/board/comment`

### method : DELETE

### comment : 게시판에 달린 댓글을 삭제하는 API

### request :
~~~json
{
    "genre" : "kpop",
    "commentNum" : "2"
}
~~~

### response :
~~~json
{
    "" : ""
}
~~~
### status code
인증 실패 : 401 (HttpStatus.UNAUTHORIZED)

성공 : 204 (HttpStatus.NO_CONTEN)

// 성공을 제외한 예외처리
실패 : 400 (HttpStatus.BAD_REQUEST)
