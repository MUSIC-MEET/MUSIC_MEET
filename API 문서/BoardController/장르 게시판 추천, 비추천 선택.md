# 장르 게시판 추천, 비추천 선택
### API명 : `/board/vote`

### method : PUT

### comment : 장르 게시판의 해당 글을 추천 혹은 비추천 눌렀을때 사용되는 API

### request :
~~~json
{
    "genre" : "kpop",
    "boardNum" : "3",
    "vote" : "upvote" or "downvote"
}
~~~

### response :
~~~json
{
    "" : ""
}
~~~
### status code
인증 실패 : 401 (HttpStatus.UNAUTHORIZED)

성공 : 204 (HttpStatus.NO_CONTENT)

// 누르는 순간에 state가 1이 되어 삭제된 글인 경우
실패 : 400 (HttpStatus.BAD_REQUEST)
