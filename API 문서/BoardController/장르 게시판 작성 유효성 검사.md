# API 제목
### API명 : `/board/validate`

### method : PUT

### comment : 글 작성시 글자수 유효성 검사를 실행함 제목은 1~20자, 내용은 1~1000자까지 유효함

### request :
~~~json
{
    "title" : "오늘은 내가 짜파게티 요리사",
    "content" : "는 개뿔 아구찜 개땡기네"
}
~~~

### response :
    NULL


### status code
가능 : 204 (HttpStatus.NO_CONTENT)

불가능 : 400 (HttpStatus.BAD_REQUEST)
