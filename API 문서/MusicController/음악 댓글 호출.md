# 음악 댓글 호출.md
### API명 : `/music/comment/{musicNum}`

### method : GET

### comment : 해당 musicNum에 달린 댓글들을 호출함

### request :
    NULL

### response :
~~~json
{
    [
        {
            "comment": "주호가 누군데 차트에 있냐",
            "user": "전국노예자랑",
            "createAt": "2022-09-23T13:22:09.000+00:00"
        }
    ]
}
~~~
### status code
호출 성공 : 200 (HttpStatus.OK)

댓글 없음 : 404 (HttpStatus.NOT_FOUND)
