# 회원가입
### API명 : `/findpw`

### method : POST

### comment : 이메일을 전달 받고 아이디를 찾는 API 

### request : 
~~~json
{
  "id" : "exid", // String  5~20자 
  "email" : "123456fF@#", // String  8~16자
  "key" : "{keyValue}"
}
~~~


### return
회원 정상 생성 : 200 (HttpStatus.OK)

정상 처리 실패 : 400 (HttpStatus.BAD_REQUEST)
